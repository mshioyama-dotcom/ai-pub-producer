# STEP0 著者プロファイル抽出 テストデータ集

Dify上で「STEP0_著者プロファイル_パターンA」「STEP0_著者プロファイル_パターンB」をテストするためのサンプルデータ。

すべて架空の著者を想定したテストデータ。実在の人物・著作とは無関係。

この本では、再現性のあるやり方だけを書いた。私が試して、続いて、結果が出たやり方だ。逆に「センスが必要」「人脈が命」みたいな話は一切書かない。なぜなら、それは普通の会社員には再現できないから。

第1章 「小さく始める」が最強の戦略

多くの副業本が「まず10万円稼ぐ計画を立てよう」と書く。私はこれに反対だ。なぜなら、計画を立てた瞬間に9割の人が動けなくなるから。

副業を始める前の私は「3年で会社を辞める」と豪語していた。でも今は違う。「会社員のままで、副収入を持つ自分」を続けている。なぜなら、その方が精神的に楽だし、本業と副業の相乗効果もあるからだ。

最初の月の収入は3,200円だった。妻に見せたら笑われた。でも私は嬉しかった。なぜなら、それは「自分の力で稼いだ初めてのお金」だったからだ。

5冊目になる本書では、これまで触れてこなかった「副業の続け方」をテーマにしました。

これからは、副業を始める人ではなく、「副業2年目以降の人」に向けた発信を増やしていきます。同じように悩んでいる人の役に立ちたいからです。

## パターンB（Agent-Chat）テストデータ

### テストケースB-1：標準的な著者ペルソナ（副業系）

**目的**：6つの質問が順番通りに出るか、6問目で【著者プロファイル】が整形出力されるかを確認。

**操作手順**：以下の回答例を順番に1ターンずつ送信。

**初回送信（インタビュー開始）**：
```
はじめます
```

**Q1（専門領域・強み）への回答**：
```
35歳から副業でWebライターを始めました。今は5年目で、副業だけで月30万円ほど稼いでいます。会社員を続けながら副業で稼ぎ続けるノウハウが、私の強みだと思います。特別なスキルはなくて、誰でもできる方法を実践してきただけです。
```

**Q2（ターゲット読者像）への回答**：
```
30代後半から40代の会社員、特に「真面目だけど目立たない」タイプの人に届けたいです。副業を始めたいけど踏み出せない、または始めたけど続かない、という悩みを持っている人。共働きで子どもがいて、平日の夜に1〜2時間しか作業できないような人をイメージしています。
```

**Q3（思想・価値観）への回答**：
```
「小さく始める」と「続けることが稼ぐことより大事」、この2つは絶対に譲れません。あと、副業を「会社を辞めるための手段」じゃなくて「会社員人生を豊かにする保険」として位置づけています。本業を犠牲にして副業をやる人は応援できません。
```

**Q4（文体・フレーズ）への回答**：
```
口癖というか、よく書く言い回しは「結論から言うと、〜」「私自身、〜」「やるかやらないか。それだけ。」みたいな、短く断定する文体です。あと「私の最初の月の収入は3,200円でした」みたいに、数字をリアルに出すことが多いです。読者を励ますために、自分の弱さを先に見せます。
```

**Q5（実績・経歴サマリー）への回答**：
```
副業歴5年、Webライターとして月収30万円。書籍は今度5冊目を出します。ブログは月間PV3万、メルマガ読者は800人くらいです。Twitterフォロワーは1.2万。一般的な会社員（営業職）として15年勤務しています。
```

**Q6（ブランドコンセプト・シリーズ方針）への回答**：
```
「再現性のあるやり方だけを書く著者」として届けたいです。シリーズで一貫させたいのは、「普通の会社員が、特別なスキルなしで、家族との時間も大事にしながら、収入の柱を増やす」というテーマ。1冊目は始め方、2冊目は単価アップ、3冊目はメンタル管理、4冊目は時間術、5冊目は続ける技術、と段階的に積み上げていく構想です。
```

**期待される最終出力**：
- 6問目の回答後、要約は出さず、直接【著者プロファイル】が整形出力される
- 6項目すべてが回答内容を反映している
- 「文体・フレーズ」項目に具体例が3つ以上含まれる

### テストケースB-2：別ペルソナ（時短家事系・短い回答）

**目的**：回答が短くても処理が進むか、深掘り質問が出ないか確認。

**初回送信**：
```
はじめます
```

**Q1への回答**（短い）：
```
時短家事です。共働きで子育て中の母親向けに発信しています。
```

**Q2への回答**：
```
20代後半〜40代の共働き・育児中のお母さん。完璧主義で疲れている人。
```

**Q3への回答**：
```
完璧主義を捨てる。家事は家族の機嫌のためにやる。
```

**Q4への回答**：
```

```

**Q5への回答**：
```
特に経歴と呼べるものはありません。普通の会社員（事務職）で、子ども2人の母親です。インスタフォロワー5000人。
```

**Q6への回答**：
```
完璧主義に苦しむお母さんに「やらないことを増やしていい」と伝える存在になりたい。シリーズ化はまだ考えていません。
```

**期待される最終出力**：
- 回答が短くても深掘り質問が出ず6問完走する

- 「ブランドコンセプト・シリーズ方針」のシリーズ部分は「（記載なし）」または「現時点では未定」と書かれる

## テスト観点チェックリスト

### パターンA共通

- [ ] 出力が【著者プロファイル】の見出しから始まる
- [ ] 6項目すべての見出し（■専門領域・強み 等）が出力される
- [ ] 6項目以外の余分な見出しが追加されていない
- [ ] 解説・前置き・後書きが一切ない
- [ ] 抽象語だけでなく、具体例・数字・固有名詞が残っている
- [ ] 「文体・フレーズ」に具体例が3つ以上含まれる
- [ ] 入力に書かれていない情報が「（記載なし）」と明示される

### パターンA-3（更新モード）特有

- [ ] 既存プロファイルの記述が無闇に削られていない
- [ ] 新素材の数字（5冊目、月30万円等）が反映されている
- [ ] 矛盾がある箇所が「過去：〜／現在：〜」形式で残されている

### パターンA-4（空欄）特有

- [ ] 警告メッセージのみが出力される
- [ ] 【著者プロファイル】が誤って生成されない

### パターンB共通

- [ ] 各ターンで質問は1つだけ
- [ ] 質問1→質問2→質問3→質問4→質問5→質問6の順で出る
- [ ] 各ターンでユーザー回答の2〜3文要約 → 次の質問、の流れ
- [ ] 深掘り質問が出ない
- [ ] 6問目の回答後、要約ではなく直接【著者プロファイル】が出る
- [ ] 出力に「ありがとうございました」等の余計な挨拶がない
